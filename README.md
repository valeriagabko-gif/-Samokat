## 🔍 Как мы работали над проектом (хакатон Samokat)

### Исходные данные от заказчика

На старте у нас было:
- **Бизнес-ценность:** рост удержания пользователей
- **Проблема:** пользователи уходят в сторонние сервисы за рецептами
- **AS IS:** рецепты только в формате сторис, архив не используется
- **Архитектура:** рецепт = медиа (обложка + слайды + подборка товаров)

### Что сделала команда (5 бизнес-аналитиков)

| Этап | Что сделали | Мой вклад |
|------|-------------|-----------|
| 1. Анализ AS IS | Изучили текущую реализацию рецептов в приложении, выявили ограничения | Участвовала в анализе текущего функционала, описала проблемные зоны |
| 2. Исследование | Провели анализ конкурентов (Яндекс Лавка, ВкусВилл, Лента, METRO) | Собрала и структурировала данные по конкурентам в Miro |
| 3. Моделирование TO BE | Построили BPMN-модель нового процесса | Помогала в описании логики подбора продуктов и замены товаров |
| 4. Формулировка требований | Разработали 18 ФТ, 10 НФТ и бизнес-требования | Отвечала за разделы: алгоритмы подбора, замена товаров, работа с корзиной |
| 5. Прототипирование | Создали LoFi-прототипы интерфейсов | Участвовала в проработке экранов корзины и карточки рецепта |
| 6. Защита | Подготовили презентацию и защитили решение | Помогала с описанием ценности решения для бизнеса |

### Что было на входе (от заказчика)

- Рецепты только в сторис
- Архив не использовался
- Нет связки «товар → рецепт»
- Нет массового добавления в корзину
- Нет отзывов и оценок

### Что получилось на выходе (TO BE)

- ✅ Отдельный раздел «Рецепты» на главном экране
- ✅ Контекстная выдача рецептов в карточке товара
- ✅ Полноценная карточка рецепта с медиа, временем, ингредиентами
- ✅ Добавление всех товаров в корзину в 1 клик
- ✅ Автоподбор товаров по тегам, популярности, наличию
- ✅ Замена отсутствующих товаров на альтернативы
- ✅ Сохранение корзины в «Избранное»
- ✅ Отзывы, оценки, средний рейтинг
- ✅ Аналитика: просмотры → корзина → заказ

### Ключевые бизнес-метрики

| Метрика | Целевое значение |
|---------|------------------|
| Удержание (CR) | +15% |
| NPS | +15% |
| Просмотры рецептов | +10% за 3 мес. |
| Средний чек | +15% |
| Частота заказов | +10% |
| Доля заказов после рецепта | 10% |
| Активные пользователи с заказом по рецепту (раз в неделю) | 20% |

### Инструменты и артефакты

- **Miro** — анализ конкурентов, мозговой штурм
- **BPMN** — модель процесса TO BE
- **Figma / LoFi** — прототипы интерфейсов
- **Google Docs** — ТЗ с требованиями
- **Презентация** — защита решения перед заказчиком
